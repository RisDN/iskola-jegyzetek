Összegzés

Adott egy N hosszúságú elemű lista. Szeretnénk, hogy ennek elemeinek az összegét vissza adja a program. Ezt könnyen megtehetjük úgy, hogy végig futunk minden egyes elemen, majd egy váltózóhoz mindíg hozzá adjuk azt az értéket, aminél tart.

Pszeudókód: 

Eljárás 
Lista
Összeg
Ciklus végig a listán
Aktuális lista elem hozzáadása az összeghez
Ciklus vége
Összeg kiírása

Példa: 

lista = [3, 8, 2, 4, 5, 1, 6] # Tömbünk
osszeg = 0 # Változó amihez mindíg hozzá adjuk az értékeket
for num in lista: # Ciklus ami végig fut minden egyes tagon
    osszeg = osszeg + num # Azt a tagot ahol tart, hozzá adja az összeghez
print("Összeg: ", osszeg) # Kiírjuk a végeredményt

Megszámolás

Adott egy N hosszúságú elemű lista. Szeretnénk, megvizsgálni hogy az elemei közt hány darab 7-es van. Végig futunk egy ciklus segítségével minden egyes elemen, és megvizsgáljuk azt, hogy 7-es e. Ha igen, egy változóhoz hozzá ad 1-et, ezzel jelezve a 7-esek darabszámát.

Pszeudókód: 

Változó
    i:Egész
    Db:=0
    Ciklus i=1-től N-ig
        Ha T(X[i]) akkor Db:=Db+1
    Ciklus vége
Eljárás vége.

Példa: 

lista = [7, 5, 4, 3, 2, 9, 7] # Listánk
i = 0 # Ciklus váltózó 0-ról indul
db = 0 # Darabszám 0-ról indul
while i < len(lista): # Ciklus amíg az I kissebb mint a listánk hossza
    if lista[i] == 7: # Amennyiben a lista I-edik tagja 7
        db += 1 # Hozzáad egyet a darabszámhoz
    i += 1 # Ciklus váltózó növelése, hogy haladjon
print("Összesen", db, "hetes van a listában.") # Végeredmény kiírása (2)

Eldöntés

Adott egy N hosszúságú elemű lista. Szeretnénk megtudni, hogy van e benne egy adott szám. Végig futunk minden egyes elemen, ha talál nekünk megfelelőt, akkor tovább halad, ha nem akkor kilép.

Pszeudókód: 

Változó
        i:Egész
        i:=1
    Ciklus amíg i≤N és nem T(X[i])
        i:=i+1
    Ciklus vége
    Van:=i≤N
Eljárás vége.

Példa: 

lista = [1, 4, 3, 2, 1, 3, 4, 5] # Listám
i = 0 # Ciklus váltózó 0-ról indul
while i<N and not T(lista[i]): # Lehet lista[i] != 4 vagy bármi
    i += 1 # Ciklus változóhoz hozzá ad egyet, hogy halodjon
van = i <= len(lista) # Amennyiben a ciklus változó kisseb vagy egyenlő
#mint a lista hossza, értékünk igaz lesz
